Data Science Showcase

Machine Learning Model for Predictive Analytics

Used Python to create and evaluate machine learning models for predicting loan defaults. Presented visualizations that explained feature importance and model performance.

Code Sample:
import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.ensemble import RandomForestClassifier
from sklearn.metrics import accuracy_score

# Load dataset
data = pd.read_csv('loan_data.csv')

# Data Preprocessing
X = data.drop('default', axis=1)
y = data['default']

# Split data
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42)

# Model Training
model = RandomForestClassifier(n_estimators=100, random_state=42)
model.fit(X_train, y_train)

# Predictions
y_pred = model.predict(X_test)

# Model Evaluation
accuracy = accuracy_score(y_test, y_pred)
print(f'Accuracy: {accuracy:.2f}')

Results: Achieved 85% accuracy in predicting loan defaults, demonstrating effectiveness of the RandomForest model for classification tasks.
